# 📚 Study Planner

A simple and interactive Study Planner built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Saves data using localStorage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
Open `index.html` in your browser.
